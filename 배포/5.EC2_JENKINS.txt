$ sudo yum -y update
$ sudo yum list java*


JDK 설치 
Amazon Corretto 란 무료로 사용할 수 있는 Open Java Development Kit (OpenJDK) 의 프로덕션용 멀티플랫폼 배포판입니다.
$ sudo yum install -y java-11-amazon-corretto.x86_64

자바 설치 확인
$ java -version


jenkins 리포지토리 추가
$ sudo wget -O /etc/yum.repos.d/jenkins.repo http://pkg.jenkins-ci.org/redhat/jenkins.repo
$ sudo amazon-linux-extras install epel

Jenkins GPG Key 추가
Jenkins를 설치할 때 신뢰할 수 있는 소스로부터 제공되는 파일이라는 것을 증명하기 위해 로컬 GPG 키링에 Jenkins GPG Key를 추가해줘야 한다.

$ sudo rpm --import http://pkg.jenkins-ci.org/redhat/jenkins-ci.org.key

Jenkins 설치
$ sudo yum install jenkins

Jenkins 서버 시작
$ sudo service jenkins start

8080 포트 LISTEN 여부 확인
$ netstat -na | grep 8080


// 메모리 상태 확인
$free -h

// swap 파일을 생성해준다. (메모리 상태 확인 시 swap이 있었지만 디렉토리 파일은 만들어줘야한다.)
$ sudo mkdir /var/spool/swap
$ sudo touch /var/spool/swap/swapfile
$ sudo dd if=/dev/zero of=/var/spool/swap/swapfile count=2048000 bs=1024

// swap 파일을 설정한다.
$ sudo chmod 600 /var/spool/swap/swapfile
$ sudo mkswap /var/spool/swap/swapfile
$ sudo swapon /var/spool/swap/swapfile

// swap 파일을 등록한다.
$ sudo vim /etc/fstab
파일이 열리면 해당 파일 아래쪽에 하단 내용 입력 후 저장
- 입력 할 수 있도록 하는 명령어 -> i
- 파일 수정 후 저장하는 명령어-> esc키 누른 후 :wq 입력 후 엔터
/var/spool/swap/swapfile    none    swap    defaults    0 0
